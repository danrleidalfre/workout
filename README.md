## Server

```bash
cd server
cp .env.example .env
npm i
docker compose up -d
npx drizzle-kit generate
npx drizzle-kit migrate
npm run seed
npm run server
```

## Web

```bash
cd web
cp .env.local.example .env.local
npm i
npm run web
```

## Mobile

```bash
cd mobile
npm i
npm run mobile
```
